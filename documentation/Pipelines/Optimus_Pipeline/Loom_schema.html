# What's in the Optimus Pipeline Loom File?

The Loom file is an HDF5 file generated using Loompy v.3.0.6 (opens new window). It contains global attributes detailing how counts were generated for the single-cell or single-nuclei parameters (Table 1). It additionally contains UMI-corrected counts as well as multiple metrics for both individual cells (the columns of the matrix; Table 2) and individual genes (the rows of the matrix; Table 3). The tables below document these metrics, list which tools generate them, and define them. This Loom file is the default matrix output of the Optimus pipeline.

NOTE

Loom files generated by Optimus are different from the final Loom file distributed on the Human Cell Atlas Data Portal (opens new window), which removes some of the metadata detailed in this document and contains additional metadata relating to each individual project.

# Table 1. Global Attributes

The global attributes in the Loom apply to the whole file, not any specific part. There are two global attributes for the Optimus Loom.

Attribute Details
LOOM_SPEC_VERSION String with the loom file spec version
expression_data_type String describing if the pipeline counts exonic or whole transcript (exonic and intronic) reads. For the single-cell mode (counting_mode = sc_rna), the value will be "exonic"; for the single-nuclei mode (counting_mode = sn_rna), the value will be "whole_transcript"
input_id The sample or cell id listed in the pipeline configuration file. This can be any string, but we recommend it be consistent with any sample metadata.

Cell Metrics Program Details

perfect_molecule_barcodes SC Tools (opens new window) The number of reads with molecule barcodes that have no errors. Metrics Definitions (opens new window)
n_mitochondrial_genes SC Tools (opens new window) The number of mitochondrial genes detected by this cell. Metrics Definitions (opens new window)
n_mitochondrial_molecules SC Tools (opens new window) The number of molecules from mitochondrial genes detected for this cell. Metrics Definitions (opens new window)
pct_mitochondrial_molecules SC Tools (opens new window) The percentage of molecules from mitochondrial genes detected for this cell. Metrics Definitions (opens new window)
reads_mapped_exonic SC Tools (opens new window) The number of reads for this entity that are mapped to exons. Metrics Definitions (opens new window)
reads_mapped_intronic SC Tools (opens new window) The number of reads for this entity that are mapped to introns. Metrics Definitions (opens new window)
reads_mapped_utr SC Tools (opens new window) The number of reads for this entity that are mapped to 3' untranslated regions (UTRs). Metrics Definitions (opens new window)
reads_mapped_uniquely SC Tools (opens new window) The number of reads mapped to a single unambiguous location in the genome. Metrics Definitions (opens new window)
reads_mapped_multiple SC Tools (opens new window) The number of reads mapped to multiple genomic positions with equal confidence. Metrics Definitions (opens new window)
duplicate_reads SC Tools (opens new window) The number of reads that are duplicates (see README.md for definition of a duplicate). Metrics Definitions (opens new window)
spliced_reads SC Tools (opens new window) The number of reads that overlap splicing junctions. Metrics Definitions (opens new window)
antisense_reads SC Tools (opens new window) The number of reads that are mapped to the antisense strand instead of the transcribed strand. Metrics Definitions (opens new window)
molecule_barcode_fraction_bases_above_30_mean SC Tools (opens new window) The average fraction of bases in molecule barcodes that receive quality scores greater than 30 across the reads of this entity. Metrics Definitions (opens new window)
molecule_barcode_fraction_bases_above_30_variance SC Tools (opens new window) The variance in the fraction of bases in molecule barcodes that receive quality scores greater than 30 across the reads of this entity. Metrics Definitions (opens new window)
genomic_reads_fraction_bases_quality_above_30_mean SC Tools (opens new window) The average fraction of bases in the genomic read that receive quality scores greater than 30 across the reads of this entity (included for 10x Cell Ranger count comparison). Metrics Definitions (opens new window)
genomic_reads_fraction_bases_quality_above_30_variance SC Tools (opens new window) The variance in the fraction of bases in the genomic read that receive quality scores greater than 30 across the reads of this entity (included for 10x Cell Ranger count comparison). Metrics Definitions (opens new window)
genomic_read_quality_mean SC Tools (opens new window) Average quality of Illumina base calls in the genomic reads corresponding to this entity. Metrics Definitions (opens new window)
genomic_read_quality_variance SC Tools (opens new window) Variance in quality of Illumina base calls in the genomic reads corresponding to this entity. Metrics Definitions (opens new window)
n_molecules SC Tools (opens new window) Number of molecules corresponding to this entity. See README.md for the definition of a Molecule. Metrics Definitions (opens new window)
n_fragments SC Tools (opens new window) Number of fragments corresponding to this entity. See README.md for the definition of a Fragment. Metrics Definitions (opens new window)
reads_per_fragment SC Tools (opens new window) The average number of reads associated with each fragment in this entity. Metrics Definitions (opens new window)
fragments_per_molecule SC Tools (opens new window) The average number of fragments associated with each molecule in this entity. Metrics Definitions (opens new window)
fragments_with_single_read_evidence SC Tools (opens new window) The number of fragments associated with this entity that are observed by only one read. Metrics Definitions (opens new window)
molecules_with_single_read_evidence SC Tools (opens new window) The number of molecules associated with this entity that are observed by only one read. Metrics Definitions (opens new window)
perfect_cell_barcodes SC Tools (opens new window) The number of reads whose cell barcodes contain no error. Metrics Definitions (opens new window)
reads_mapped_intergenic SC Tools (opens new window) The number of reads mapped to an intergenic region for this cell. Metrics Definitions (opens new window)
reads_mapped_too_many_loci SC Tools (opens new window) The number of reads that were mapped to too many loci across the genome and as a consequence, are reported unmapped by the aligner. Metrics Definitions (opens new window)
cell_barcode_fraction_bases_above_30_variance SC Tools (opens new window) The variance of the fraction of Illumina base calls for the cell barcode sequence that are greater than 30, across molecules. Metrics Definitions (opens new window)
cell_barcode_fraction_bases_above_30_mean SC Tools (opens new window) The average fraction of Illumina base calls for the cell barcode sequences that are greater than 30, across molecules. Metrics Definitions (opens new window)
n_genes SC Tools (opens new window) The number of genes detected by this cell. Metrics Definitions (opens new window)
genes_detected_multiple_observations SC Tools (opens new window) The number of genes that are observed by more than one read in this cell. Metrics Definitions (opens new window)
reads_unmapped SC Tools (opens new window) Reads that are non-transcriptomic
emptydrops_FDR dropletUtils (opens new window) False Discovery Rate (FDR) for being a non-empty droplet; not included for the sn_rna mode
emptydrops_IsCell dropletUtils (opens new window) Binarized call of cell/background based on predefined FDR cutoff; not included for the sn_rna mode
emptydrops_Limited dropletUtils (opens new window) Indicates whether a lower p-value could be obtained by increasing the number of iterations; not included for the sn_rna mode
emptydrops_LogProb dropletUtils (opens new window) The log-probability of observing the barcode’s count vector under the null model; not included for the sn_rna mode
emptydrops_PValue dropletUtils (opens new window) Numeric, the Monte Carlo p-value against the null model; not included for the sn_rna mode
emptydrops_Total dropletUtils (opens new window) Numeric, the total read counts for each barcode; not included for the sn_rna mode
